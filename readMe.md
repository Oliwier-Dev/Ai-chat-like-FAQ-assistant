## How it works

- All the questions and answers are stored in `kb.JSON`.  
- The script matches your question with the closest keywords in the JSON and shows the answer.  
- You can customize it by editing the text inside `kb.JSON`.  

**Note:**  
Do **not** change the individual keys like `id`, `keywords`, `question`, or `answer`. Only change the text inside otherwise, the code may break.

---

## How to use

1. Open the live demo link or run it locally by opening `index.html`.  
2. Type your question in the input box.  
3. The assistant will show the most relevant answers from the knowledge base.
